# @piyoppi/counter-button

Custom elements that increment counts, display counts

## Usage

### Example

```html
<counter-container
  apiurl="https://api.example.com"
  >
  <counter-increment-button>
    <span>
      &#x1f31f;
      <counter-display>
        <span slot="loading">...</span>
      </counter-display>
    </span>
  </counter-increment-button>
</counter-container>
```
